# BakShell

A blazing-fast, customizable shell with Lua plugin support — rewritten in Go.

Zero system dependencies. Single static binary, ~3MB stripped.

```sh
go build -ldflags="-s -w" -o bsh ./cmd/bsh
./bsh
```
